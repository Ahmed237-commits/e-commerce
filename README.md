# My Next Shop

My Next Shop is a simple e-commerce website built using HTML, CSS, and JavaScript. The project focuses on creating a clean and responsive shopping experience where users can browse products and interact with a modern user interface.

---

## Features

* Product listing section
* Responsive design for different screen sizes
* Product cards with images and details
* Shopping cart functionality
* Interactive user interface using JavaScript
* Clean and organized layout

---

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
